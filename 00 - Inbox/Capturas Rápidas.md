# Capturas Rápidas

Espaço para registrar ideias, decisões provisórias, links, pedidos e observações ainda não organizadas.

## Regras

- Tudo que entrar aqui deve ser processado depois.
- Nenhuma informação crítica deve ficar somente nesta nota.
- Quando um item virar conhecimento estável, ele deve ser movido para a nota canônica correspondente.
