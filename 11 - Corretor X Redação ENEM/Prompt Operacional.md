# Prompt Operacional

Status: rascunho operacional

## Papel

Este arquivo aponta para o prompt-base do Corretor X e registra suas regras de uso.

## Princípios

- O prompt deve seguir [[Protocolo de Correção]].
- A correção deve usar [[Manual Oficial ENEM 2025]] como autoridade.
- [[Padrões Nota 1000]] entram apenas como referência interpretativa.
- O modelo deve declarar incerteza quando faltar informação.

## Campos obrigatórios

- Tema completo.
- Textos motivadores ou resumo autorizado.
- Redação integral.
- Observações de legibilidade, cópia ou atendimento especializado.
